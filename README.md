# 🟩 Minecraft Guilin City Walk | 桂林像素漫步 🏔️

这是一个将 **桂林山水** 与 **Minecraft (MC) 像素艺术风格** 与**TRAE**完美融合的城市漫步导览网站。

[![Deploy to GitHub Pages](https://github.com/MU-ty/guilin-travel-and-trae/actions/workflows/deploy.yml/badge.svg)](https://github.com/MU-ty/guilin-travel-and-trae/actions/workflows/deploy.yml)

## 🌟 项目愿景
我们致力于构建一个充满美好与科技的山水桂林。在这里，每一条旅游路线都是一段方块冒险，每一个景点都是一个像素故事。

> “你不必拥有技术背景，只要你拥有想法即可。” —— 史蒂夫 (MU-ty)

## 🛠️ 核心功能
- **PixelMap (像素地图)**: 基于 SVG 实现的交互式放射状像素路径图，涵盖市中心、阳朔、龙脊等核心景区。
- **TRAE 在桂林**: 记录 TRAE 社区在桂林的开发者活动（Meetup、Demo Day），支持资料下载与地图标记。
- **MC 风格 UI**: 全站采用 `Press Start 2P` 像素字体，模拟游戏界面的方块化按钮、阴影与材质。
- **管理员系统**: 史蒂夫（管理员 MU-ty）的个人空间，支持 Coffee Chat 预约与 GitHub 深度互动。

## 🚀 技术栈
- **框架**: [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **样式**: [Tailwind CSS](https://tailwindcss.com/) (自定义 MC 材质色系)
- **动画**: [Framer Motion](https://www.framer.com/motion/) (丝滑的 UI 弹窗与地图交互)
- **部署**: GitHub Actions 自动化构建与部署

## 📦 本地开发
1. 克隆仓库:
   ```bash
   git clone https://github.com/MU-ty/guilin-travel-and-trae.git
   ```
2. 安装依赖:
   ```bash
   npm install
   ```
3. 启动开发服务器:
   ```bash
   npm run dev
   ```
